# Related Work

## Positioning against prior work

| Method | Venue | Relation to ET-COME |
|--------|-------|-------------------|
| **Conformalised Data Synthesis** | ML 2025, arXiv:2312.08999 | Uses ICP to identify where to synthesise. ET-COME uses OT for placement and OOB screening for admission. Complementary: they handle "where" with CP; we handle "whether" with CP and "where" with risk-targeted OT. |
| **Filtering with Confidence** | arXiv:2509.21479, 2025 | Conformal filtering of generative outputs. Not imbalance-specific, not OOB-based. Closest structural relative to Module C outside imbalanced learning. |
| **UQDIR** | Expert Systems 2024 | Epistemic UQ guiding resampling in imbalanced regression. No formal aleatoric/epistemic decomposition, no synthesis, no OT. Closest relative to Module A. |
| **LEO-CVAE** | arXiv:2509.25334, 2025 | Entropy-guided CVAE for uncertain regions. Shannon entropy proxy only, CVAE generator, no OOB, no OT. Related in intent to Modules A and B. |
| **P²OT** | ICLR 2024, arXiv:2401.09266 | Sinkhorn-type matrix scaling for imbalanced clustering. Not supervised synthesis, not tabular classification. The only prior Sinkhorn-type work in imbalanced learning. |
| **MWMOTE** | *IEEE TKDE* 26(2), 2014 | Distance-weighted minority oversampling. Synthesis targets are weighted by local minority density and proximity to the boundary — similar in intent to Module A's synthesis selection but uses geometric distance rather than ensemble epistemic uncertainty; no OT placement; no conformal screening. Closest density-guided structural relative to Module A. |
| **ADASYN** | IJCNN 2008 | Density-adaptive synthesis; generates proportionally more samples in regions where minority density is low relative to majority. Targets high-overlap regions by design, which Module A explicitly excludes via the aleatoric filter. ADASYN is Tier 1 baseline; included here to clarify positioning. |
| **SOMO** | *Expert Systems with Applications* 82, 2017 | Self-Organizing Map oversampling; learns minority manifold topology via unsupervised dimensionality reduction. No ensemble feedback, no OT, no conformal screening. Closest unsupervised structural relative to Module B's topology-aware placement. |
| **CGMOS** | CIKM 2016, arXiv:1607.06525 | Certainty-Guided Minority OverSampling. Uses posterior class certainty change to SELECT WHERE synthetic minority points are placed — the direct conceptual predecessor to Module A's epistemic targeting. Key differences: certainty is from a generic fitted classifier, not OOB-derived; no epistemic/aleatoric decomposition; no OT placement; no conformal screening. |
| **OTOS** | AAAI 2019 | Oversampling via Optimal Transport. Uses an explicit OT coupling to place synthetic minority points matching the global minority distribution; OTOS-β shifts synthesis toward high-loss examples. The only prior paper using a true OT transport plan for minority synthesis placement on tabular data — direct predecessor to Module B. Key differences: no ensemble uncertainty, no OOB, no conformal screening; loss signal (not epistemic uncertainty) shifts the synthesis target. |
| **Shaker & Hüllermeier (2020)** | DS 2020, arXiv:2001.00893 | Formal framework for epistemic/aleatoric decomposition in random forests using inter-tree variance (epistemic) and intra-leaf label distributions (aleatoric). Establishes the decomposition framework that Module A applies. Critical gap: purely an uncertainty *estimation* paper — the decomposition has never been applied to synthesis targeting or imbalanced learning, and uses ensemble variance rather than OOB predictions. |
| **D'souza et al.** | AAAI 2025, arXiv:2412.15657 | Uses k-fold RF cross-validation to identify minority/majority overlap region; synthesises into that region and under-samples overlapping majority points. Closest prior work to Module A's concept of using an RF signal to identify synthesis target regions. Key differences: uses held-out CV folds (not OOB), identifies overlap rather than decomposed epistemic uncertainty, no formal uncertainty decomposition, no OT, no conformal screening. |
| **SMOGAN** | arXiv:2504.21152, 2025 | SMOTE generates candidates; a trained GAN discriminator (trained on real minority data) rejects out-of-distribution synthetic candidates. Structurally identical role to Module C — post-generation admission screening of synthetic candidates. Key differences: uses GAN discriminator (not OOB ensemble), requires training a generative model, no OT-based placement, screening is not class-calibrated. |
| **RBO / MC-RBO** | *Neurocomputing* 2019; IJCAI 2019 | Radial-Based Oversampling uses Gaussian RBF potential fields to synthesise minority samples where mutual class potential ≈ 0 (maximum overlap zone). A geometry-based proxy for classification uncertainty guides synthesis placement. No ensemble feedback, no OOB, no formal uncertainty decomposition, no OT, no conformal screening. |

## Empirical novelty controls (not prior art — evaluation hooks)

Related work establishes **component-level** predecessors; it does not resolve whether **shared-OOB coupling** or **entropy partitioning** beats simpler pipelines. The plan addresses that outside this table: **Baseline-A-simple** (ϕ_total-only vs. decomposition), **Baseline-Sequential** with **E_A = T₀** (fair) plus **equal-total** secondary control, **Criterion 0** (vs. class-weighted ExtraTrees), correlation diagnostic **in main paper** ([08-evaluation.md](08-evaluation.md)), and the **2D synthetic** sanity figure — see [07-ablation.md](07-ablation.md) and [08-evaluation.md](08-evaluation.md).
