# ET-COME

Equilibrium Transport with Conformal Minority Expansion for class-imbalance research.

## Install

Install from PyPI:

```bash
pip install et-come
```

Install from source (recommended for contributors):

```bash
git clone https://github.com/Pawlo77/ET-COME.git
cd ET-COME
make install
```

## Quick Start

Minimal import check:

```python
import et_come

```

## Project Structure

- `src/et_come/` - Python package source
- `tests/` - test suite
- `docs/` - Sphinx documentation
- `experiments/` - research and experiment assets
- `examples/` - usage examples
- `report/` - report assets/templates
